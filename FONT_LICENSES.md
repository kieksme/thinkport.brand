## SIL Open Font License 1.1 Summary

The SIL Open Font License (OFL) is a free, libre and open source license specifically designed for fonts and related software based on our experience in font design and linguistic software tool development.

### Key Points

1. **Free to Use:** Fonts can be used for any purpose, including commercial projects
2. **Modification Allowed:** Fonts can be modified and distributed
3. **No Warranty:** Fonts are provided "as is" without warranty
4. **Attribution:** Original copyright notices must be preserved
5. **No Selling:** Fonts cannot be sold by themselves (but can be included in products)

For the complete license text, visit: <https://scripts.sil.org/OFL>
