Overview

Portfolio PDFs are one-page documents generated for Thinkport staff. They include a branded header with background graphic and logo, personal info, skills with experience bars, certificates (with issuer, level, issue date), optional sections for education, description and booking link, and a consistent layout using Montserrat and brand colors.

Content

  • Header with background SVG and Thinkport logo
  • Avatar (circular, white border, semi-transparent background)
  • Name, position, email (linked), experience
  • Skills & Erfahrung (horizontal bars)
  • Zertifikate (compact multi-column cards)
  • Optional: Ausbildung, Über [Vorname], Termin buchen

Technical

  • Generated via scripts/generate-portfolio-pdf.mjs (HTML + Puppeteer)
  • Data from Thinkport API (Thinkport staff only)
  • Assets (background, logos) embedded as data URIs for reliable PDF rendering
  • Output: release-assets/staff/portfolios/portfolio-<slug>.pdf
