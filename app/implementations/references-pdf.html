Overview

Dieses Tool lädt Referenzen (Case Studies) aus der Thinkport GraphQL References API und erzeugt daraus eine mehrseitige PDF im gleichen A4-Briefpapier-Raster wie A4 Letterhead. Filter nach Kategorien und Tags, wähle einzelne Referenzen aus und exportiere lokal im Browser (jsPDF + html2canvas).

Ablauf

  1. Zugangsdaten eingeben (Basic Auth, wie im API-Playground)
  2. „Referenzen laden“
  3. Kategorien/Tags einschränken (leer = alle)
  4. Referenzen auswählen, „PDF generieren“

Hinweise

  • Lokal: optional THINKPORT_REFERENCES_USER / THINKPORT_REFERENCES_PASS in .env für den Vite-Proxy (pnpm dev).
  • Hero-Bilder brauchen CORS auf der Bild-URL, sonst kann der Export abbrechen.
  • Langer Text wird auf einer Seite begrenzt (Overflow hidden), analog zur ersten Ausbaustufe.
