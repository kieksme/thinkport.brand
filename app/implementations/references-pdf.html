Overview

Dieses Tool lädt Referenzen (Case Studies) aus der Thinkport GraphQL References API und erzeugt daraus eine mehrseitige PDF im gleichen A4-Briefpapier-Raster wie A4 Letterhead. Filter nach Kategorien und Tags, wähle einzelne Referenzen aus und exportiere als PDF (im Browser mit jsPDF + html2canvas, oder auf einer Netlify-Deployment-Variante serverseitig).

Netlify-Modus: Basic Auth für die Thinkport-API liegt in THINKPORT_API_USERNAME / THINKPORT_API_PASSWORD. Aufrufe laufen über /api/thinkport/…. Die Felder unten sind ausgeblendet. Siehe NETLIFY.md.

Ablauf

  1. Zugangsdaten eingeben (Basic Auth, wie im API-Playground)
  2. „Referenzen laden“
  3. Kategorien/Tags einschränken (leer = alle)
  4. Referenzen auswählen, „PDF generieren“

Hinweise

  • Lokal: THINKPORT_API_USERNAME / THINKPORT_API_PASSWORD in .env für den Vite-Proxy (pnpm dev). Netlify: siehe NETLIFY.md.
  • Motive liegen auf dem API-Host, z. B. https://thinkportapi.netlify.app/images/references/reference-simpl.png (Feld image). Ohne CORS blockiert der Browser fetch von einer anderen Origin; lokal mappt pnpm dev auf /api/thinkport/…. Auf Netlify mit VITE_THINKPORT_API_PROXY=true dasselbe. Auf rein statischem Hosting ohne Proxy: CORS oder eigener Proxy nötig.
  • Export: Karten-Listing (Orientierung thinkport.digital/referenzen), bis zu 3 Referenzen pro PDF-Seite mit Motiv, Teaser und Tags. Bilder und Logo werden vor dem Rendern verkleinert, damit der Browser weniger Pixel verarbeiten muss.

API & Filter

Kategorien

Keine Auswahl = alle Kategorien.

Tags

Keine Auswahl = alle Tags.

Referenzen exportieren

Im PDF enthalten: 0 Referenzen · ca. 0 Seiten (nur angehakte Einträge · Listing, 3 pro Seite)
